# Factors Influencing Students Academic Performance

**Autor:** João Pedro Freire Barroso Magalhães  
**Data:** 19/02/2025

---

## Introdução

Este projeto analisa os fatores que influenciam o desempenho acadêmico de estudantes, considerando notas de matemática, leitura e escrita.  
O dataset inclui informações sobre gênero, nível de escolaridade dos pais, tipo de almoço e participação em cursos preparatórios.  
O objetivo é identificar padrões que possam fornecer insights para educadores e gestores.

---

## Perguntas Analíticas

O projeto busca responder perguntas como:  
- Existe diferença de desempenho entre gêneros?  
- O nível de escolaridade dos pais está relacionado a notas mais altas?  
- Estudantes que participaram de cursos preparatórios apresentam melhores resultados?  
- O tipo de almoço está associado a diferenças no desempenho acadêmico?  
- Qual disciplina apresenta maior variação nas notas?

---

## Dataset

- **Fonte:** [Students Performance Dataset Cleaned — Kaggle](https://www.kaggle.com/datasets/muhammadroshaanriaz/students-performance-dataset-cleaned/data)  
- Total de registros: 1000 estudantes  
- Colunas principais:  
  - `gender`  
  - `race_ethnicity`  
  - `parental_level_of_education`  
  - `lunch`  
  - `test_preparation_course`  
  - `math_score`  
  - `reading_score`  
  - `writing_score`

---

## Visualizações e Insights

- **Notas por gênero:** Meninos têm média maior em matemática; meninas têm média maior em leitura e escrita.  
- **Curso preparatório:** Estudantes que participaram do curso apresentam médias mais altas em todas as matérias.  
- **Educação dos pais:** Quanto maior o nível de escolaridade dos pais, maior o desempenho médio dos alunos.  
- **Tipo de almoço:** Alunos com almoço padrão apresentam desempenho superior.  
- **Variação das disciplinas:** Matemática e escrita apresentam maior dispersão em comparação com leitura.
